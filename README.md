# 🎯 YouTube Comment Sentiment Analyzer

Analyze the sentiment of YouTube comments in real-time!  
Visualize whether comments are **positive** or **negative** and gain insights quickly.  

---

## 🚀 Live Demo

[![Open in Replit](https://img.shields.io/badge/Open%20in-Replit-blue?logo=replit&logoColor=white)](https://youtube-comment-sentiment-analyzer--AbhishekShakya5.replit.app)  
Try the app now on Replit and see it in action!

---

## 🔥 Features

- ✅ Real-time sentiment analysis of YouTube comments  
- ✅ Interactive and user-friendly interface  
- ✅ Stores data securely in a production database  
- ✅ Autoscale deployment (2 vCPU / 4 GiB RAM)  

---

## 🛠️ Tech Stack

- **Backend:** Python / FastAPI
- **Frontend:** React / MUI
- **Database:** Postgres / Replit Production Database  
- **Deployment:** Replit Autoscale  

---

## 📚 How to Use

1. Open the live app via the badge above.  
2. Enter a YouTube video link or comment data.  
3. View the sentiment breakdown instantly!
